# closr
A monorepo to build a closely connected social graphs based on contexts

# set it up
Commands are in reference to the root dir

* Install client deps
`cd client`
`npm install`

* Install server deps
`cd server`
`npm install`

* Run
`npm run dev`


# push to heroku
`cd server`
`git commit -am "your changes"`
`git push heroku master`
